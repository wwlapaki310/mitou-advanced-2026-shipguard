# ShipGuard 🚢
### 小型船舶免許保持者向け操縦支援スマートフォンアプリ

**未踏アドバンスト事業 2026年度上期 提案プロジェクト**

---

## 概要

日本の小型船舶免許保持者は約270万人に達する一方、免許取得のハードルが低く実技経験が乏しいまま操縦する人が多い。船舶事故の約8割を小型船が占め、プレジャーボートによる事故は増加傾向にある。

本プロジェクトでは、**追加ハードウェア不要のスマートフォン単体**で動作する操縦支援アプリ「ShipGuard」を開発する。既存のあらゆる船舶に「0円で後付け」できる安全支援基盤の実現を目指す。

---

## 背景・課題

| 指標 | 内容 |
|------|------|
| 年間船舶事故件数 | 約2,000件/年 |
| 小型船舶の占有率 | 全体の約8割 |
| プレジャーボートの動向 | 漁船は減少傾向、**プレジャーボートは増加傾向** |
| 免許保有者数 | 約260〜290万人 |
| 免許取得の実態 | 半日講習＋約1時間の実技・筆記で取得可能 |
| 練習機会 | レンタル費用が高く自主練習が困難 |
| 既存自動化技術 | LiDAR＋カメラは全船舶への導入が非現実的 |

---

## 3つのコア機能

### ① 高速外界検知モジュール
- スマホカメラを用いたリアルタイム障害物検知
- 軽量CV手法（YOLOv8n等）でFPSを最優先
- 船舶・浮遊物・人・岸壁の分類＆距離推定
- VLMを使わず古典的・量子化モデルで実用速度を確保

### ② 姿勢・危険度モニタリング
- スマホ内蔵ジャイロ・加速度センサーの活用
- 横揺れ角度・急加速の定量化
- 転覆危険度スコアのリアルタイムHUD表示
- センサーフュージョンによる客観的な危険判定

### ③ ローカルLLM操船相談
- オフライン動作（Gemma / Qwen等の量子化モデル）
- 海事法規・気象・操船ルールの自然言語相談
- 海事知識ベースを用いたRAG構成
- 電波の届かない海上でも動作

## 未踏性（既存技術との差分）

| 比較対象 | 課題 | 本提案の優位性 |
|----------|------|---------------|
| LiDAR＋カメラ自動操縦 | 数十万円の専用HW＋工事が必要 | スマホ単体・追加HW不要 |
| 一般ナビアプリ | 障害物検知・姿勢センサー・LLMなし | 3機能を統合 |
| VLMベース海上認識 | FPSが出ず実用に耐えない | 軽量CV手法でFPS最優先 |
| 個別機能アプリ | 検知/センサー/LLMが分断 | 単一アプリで統合 |
