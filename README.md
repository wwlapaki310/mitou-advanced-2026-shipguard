# 小型船舶操縦士向け操縦補助アプリケーションの開発

**未踏アドバンスト事業 2026年度上期 提案プロジェクト**

---

## 概要

日本の小型船舶免許保有者は約270万人に達する一方、スクールを利用した場合でも1日の学科・実技講習と翌日午後の国家試験という短期間で取得可能で、実技経験が乏しいまま操縦するケースも多い。船舶事故の約8割を小型船が占め、プレジャーボートによる事故は増加傾向にある。

本プロジェクトでは、**追加ハードウェア不要のスマートフォン単体**で動作する操縦補助アプリケーションを開発する。既存のあらゆる船舶に「0円で後付け」できる安全支援手段の実現を目指す。

---

## 5つのコア機能

| # | 機能 | 装着モード | 技術 |
|---|------|-----------|------|
| ① | リアルタイム外界検知 | 船体固定 | 軽量物体検知モデル（TFLite/CoreML）|
| ② | 転覆危険度モニタリング | 船体固定 | IMU + Complementary Filter |
| ③ | ローカルLLM操船相談 | 両モード（停泊中） | llama.cpp + RAG（完全オフライン）|
| ④ | 視線連動検知・音声通知 | おでこ装着 | カメラ + 音声優先度キュー |
| ⑤ | 見張り行動モニタリング | おでこ装着 | IMU（頭部Yaw回転）|
