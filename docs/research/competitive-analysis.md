## 3. 学術研究・データセット調査

### 海上物体検知（Maritime Object Detection）

| 研究・データセット | 内容 | スマホエッジ適用 |
|---|---|---|
| Singapore Maritime Dataset (SMD) | 海上監視映像・船舶/人/浮遊物ラベル | ❌ GPU前提 |
| SeaShips Dataset | 中国沿岸7クラス・6000+画像 | ❌ |
| MARVEL Dataset | マルチモーダル海上データ（2024） | ❌ |
| YOLOv8 海上適用研究（2023〜） | 精度面で実用水準に到達 | ❌ スマホ動作未検証が多い |
| VLM（GPT-4V等）海上認識研究 | 精度高だがFPS 5未満 | ❌ リアルタイム不可 |

→ **「FPS最優先 × スマホオンデバイス × 船舶特化クラス」の組み合わせは空白地帯**

### IMUベース姿勢検知

| 研究 | 内容 | スマホ民生IMU適用 |
|---|---|---|
| 船舶向けIMU研究（学術） | FOG・高精度MEMSを前提 | ❌ 専用HW前提 |
| Complementary Filter | ジャイロ+加速度の融合手法・確立済み | ✅ スマホで動作可能 |
| Madgwick Filter | より精度高め・計算量増 | ✅ スマホで動作可能 |
| スマホIMU転覆危険度アプリ | **存在しない** | — |

## 5. 結論：未踏性の3つの根拠

| 根拠 | 説明 |
|---|---|
| **① 機能統合の新規性** | カメラ検知 × IMU危険度 × オフラインLLMを1アプリで統合した先行例なし |
| **② 非AIS障害物へのアプローチ** | 既存アプリはAIS登録船しか扱えない。浮遊物・転覆危険など実際の事故原因をカバーできていない |
| **③ 0円後付けモデル** | 追加HWなしで270万人の既存免許保有者に即日リーチできる唯一のアプローチ |
